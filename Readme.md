# mylog.sh
Writes developers diary
# Install

`ln -s <script_path> /usr/local/bin/lg`

# Usage
```
lg - Write new entry in mylog
lg -t - today, show today file
lg -a - all, show all files
lg -c - concat, concat all files
```

- [x] Write new Log entries
